A desktop tolerance query tool based on the GB/T 1800.1-2009 standard, designed for mechanical designers and manufacturing engineers — no more flipping through handbooks to look up tolerance tables.

Key Features

Single Query: Select hole or shaft, enter the nominal size, choose the deviation symbol and tolerance grade — instantly outputs upper deviation, lower deviation, tolerance band width, and limit dimensions
Hole-Shaft Fit Analysis: Enter parameters for both hole and shaft simultaneously, automatically determines the fit type (Clearance Fit / Transition Fit / Interference Fit), and calculates the maximum/minimum clearance or interference
Query History: Automatically saves query history for easy reference and comparison
Coverage

Nominal size range: 0 ~ 500 mm
Tolerance grades: IT01 ~ IT18 (20 grades total)
Deviation symbols: Holes A ~ Z, Shafts a ~ z (including compound symbols CD, EF, FG, etc.)
